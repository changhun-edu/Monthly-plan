세 가지가 달라집니다

반복 작업을 자동화해서 선생님이 수업에 집중할 수 있도록 만들었습니다.

📋
월중계획 양식 자동 생성
월을 선택하면 달력 양식이 자동으로 만들어집니다. 토·일·공휴일은 노란색, 방학 기간은 자동 표시됩니다. 7~8월, 12~1월은 한 시트에 합산 생성됩니다.
클릭 한 번으로 완성
✅
이름 입력 → 날짜 자동 기록
확인자 칸에 이름을 입력하면 확인 일시가 자동으로 기록됩니다. 한 번 확인하면 ✅로 고정되어 다른 사람이 실수로 지울 수 없습니다.
공유 문서에서도 안전하게
🔄
월중계획 → 주간계획 자동 복사
주간계획 양식을 생성하면 월중계획에 입력한 내용(작은학교·업무)이 해당 날짜에 자동으로 들어갑니다. 같은 내용을 두 번 입력하지 않아도 됩니다.
이중 입력 완전 제거
📌
매월 반복 행사 자동 입력
청렴·갑질 자가진단의 날(매월 1일), 안전점검의 날(매월 4일), 청렴 Day(셋째 주 수요일) 등을 한 번 설정하면 월중계획 생성 시 자동으로 반영됩니다. 쉬는 날이면 다음 평일로 자동 순연됩니다.

FAQ

무료인가요?▽
완전 무료입니다. Google 계정만 있으면 Google Sheets와 Apps Script 모두 무료로 사용 가능합니다.
사본을 만들었는데 [📅 월중계획&주간계획] 메뉴가 안 보입니다.▽
편집자 권한으로 공유된 링크에서 사본을 만들어야 메뉴(스크립트)가 함께 복사됩니다.

뷰어 권한 링크에서 사본을 만들면 시트 데이터만 복사되고 스크립트는 포함되지 않습니다.

해결 방법: 이 페이지의 [문서 받기] 버튼 → [파일] → [사본 만들기] 순서로 다시 시도해 주세요.
사본 저장 후 원본 문서와 연결이 되나요?▽
아닙니다. 사본을 만들면 완전히 독립된 파일이 됩니다. 원본 문서와 데이터가 공유되지 않고 내 드라이브에서 독립적으로 관리됩니다.
"Google이 확인하지 않은 앱" 경고가 뜨는데 안전한가요?▽
안전합니다. 개인이 만든 Apps Script에 항상 표시되는 정상적인 경고입니다. 본인의 Google 드라이브에만 접근하고 외부 서버로 데이터를 전송하지 않습니다.
[고급] → [안전하지 않은 페이지로 이동] → [허용] 순서로 진행하시면 됩니다.
여러 선생님이 동시에 입력할 수 있나요?▽
네. 완성된 월중계획 파일을 편집자 권한으로 공유하면 여러 선생님이 동시에 접속해 내용을 입력할 수 있습니다. Google 계정 로그인 없이 링크만으로도 편집이 가능합니다.
월중계획을 먼저 입력해야 주간계획이 자동으로 채워지나요?▽
네. 월중계획 시트의 작은학교·업무 칸에 내용을 입력한 후 주간계획 양식을 생성하면 해당 날짜 내용이 자동으로 복사됩니다. 월중계획을 먼저 작성하는 것이 기본 업무 흐름입니다.
공휴일 자동 조회가 안 될 때는 어떻게 하나요?▽
학교 Google Workspace 계정에서 외부 API 접근이 차단된 경우 자동 조회가 안 될 수 있습니다. 초기 설정의 공휴일 입력란에 직접 입력하세요.
입력 형식: 3/1, 5/5, 6/6, 8/15, 10/3, 10/9, 12/25
